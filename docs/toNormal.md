# Модуль toNormal.py

## Описание
OCR-конвертер для PDF:
- Обрабатывает сканированные PDF
- Использует OCRmyPDF
- Сохраняет текст в поисковом формате

## Зависимости
```bash
pip install ocrmypdf
```
## Использование
```bash
python toNormal.py
```
### Пути
Вход: 
```python
inpath = "./files/list"
```
Выход:
```python
outpath = "./files/GOOD"
```
